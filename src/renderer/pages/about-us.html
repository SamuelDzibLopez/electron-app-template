Conoce más sobre este proyecto

Esta aplicación en Electron es un proyecto personal enfocado en una arquitectura limpia, comunicación IPC modular y una estructura escalable. Está inspirada en stacks modernos como React y Next.js, pero construida utilizando únicamente HTML, CSS y JavaScript.

El objetivo es mantener las cosas simples, claras y fáciles de mantener, evitando abstracciones innecesarias.
