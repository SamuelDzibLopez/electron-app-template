About the project.

Conoce más sobre este proyecto:

Electron App Template es un template base en Electron diseñado para crear aplicaciones de escritorio de forma rápida, ordenada y escalable.

Está enfocado en una arquitectura limpia, comunicación IPC modular y buenas prácticas desde el inicio.

Se inspira en stacks modernos como React y Next.js, pero utiliza únicamente HTML, CSS y JavaScript, manteniendo el desarrollo simple, claro y fácil de mantener.
